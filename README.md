# Primo TypeScript Project

A TypeScript implementation of the merge sort merge step — takes two sorted arrays and merges them into a single sorted array.

## Setup

```bash
npm install
```

## Run

```bash
npx tsc && npm start
```

This compiles the TypeScript source in `src/` to `dist/`, then runs `dist/index.js`.

**Example output:**
```
Merged Array1: [1, 2, 3, 4, 5, 6]
```

## Test

```bash
npm test
```

## Project Structure

```
src/
  index.ts   — entry point, demo of merge()
  merge.ts   — merge function for two sorted number arrays
tests/       — Jest test suite
dist/        — compiled output (generated by tsc)
```
